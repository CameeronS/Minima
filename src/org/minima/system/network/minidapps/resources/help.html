MINIDAPPS
                  Powered by Minima

Creating a MiniDAPP is very easy.

They are simply static web sites, using standard HTML, JavaScript and CSS style sheets, coupled with the minima.js lib for interacting with the Minima network.

All the files are placed in a folder, zip it, and change the extension to '.minidapp'.

You can open and read any MiniDAPP you already have to see how it works. Just change the extension to .zip and you can view them like any other archive.

You can freely distribute MiniDAPPs, without the need to host them on costly centralised websites.

Without the worry of wondering if the website will be taken down..
   if the developers will change your favourite DAPP..
   if certain countries will be censored..
   ..or even if the service will be stopped altogether.

The minima.js file is available from mifi.minima.global..

MiFi is the integration of Minima into Finance.

You can integrate Minima into your current website or work-flow.

You may want to accept payment, track and trace goods, distribute custom tokens that represent tickets or other voucher systems, or use Minima for managing in-game tokens etc etc.. Anything is possible.

A MiniDAPP is different. A MiniDAPP is when you integrate directly into Minima. The applications are stand-alone and can be run by anyone, anywhere. They allow for truly decentralised applications that do not that rely on ANY central party or service.

MiniDAPPs run directly from the Minima client application.

A simple web-server and a little custom integration result in a completely decentralised, uncensored, unfiltered, unstoppable Ecosystem for Applications.
