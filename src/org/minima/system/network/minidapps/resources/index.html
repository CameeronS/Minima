What is a DAPP ?

DAPPS are front-ends to Smart Contracts.

Smart Contracts are programs that run on Minima.

Smart Contracts can be hard to interact with.

DAPPS make interacting with Smart Contracts easy.


Given a functioning Minima network..

1) No one can stop you using it

2) No one can censor who you use it with

3) No one can force a change to it

4) No one can turn it off


If your DAPP doesn't do this - it's not a DAPP.
The Evolution will not be Centralised
